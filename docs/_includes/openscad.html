{%- comment -%} openscad.html — embed a self-hosted OpenSCAD model in a docs page. Poster-first and tiered. By default it shows a lightweight pre-rendered model: a PNG poster you click to load a ~0.6 MB rotatable 3D view (no WASM compiler). Optionally offer a per-embed "Customize" upgrade to the live in-browser customizer with live="", or embed a live surface directly with surface="customizer". Every surface is assembled into THIS site by the openscad-web GitHub Action (.github/workflows/pages.yml + openscad-publish.yml) and served SAME-ORIGIN at /models// — nothing loads from an external host. Usage: {% include openscad.html model="demo" %} {# static, poster-first #} {% include openscad.html model="demo" live="demo-live" %} {# + "Customize" upgrade #} {% include openscad.html model="demo-live" surface="customizer" %} {# live directly #} Parameters: model (required) mount name — a target's mountPath: /models// surface "static" (default) | "viewer" | "customizer" — how was published live optional mount name of a live surface, for a "Customize" upgrade button height iframe / viewer height in px (default: 480) title accessible title / label (default: the model name) The surfaces exist only in the CI-assembled site, so a plain local `jekyll serve` shows the poster/link fallback instead of a live view. {%- endcomment -%} {%- assign osc_model = include.model -%} {%- if osc_model == nil or osc_model == "" -%}
