# Installation

State what a user needs to run your library:

- **OpenSCAD** — the minimum version you support (e.g. 2021.03 or newer).
- **Dependencies** — any other SCAD libraries, and where to put them so
  `use <lib.scad>` / `include <lib.scad>` resolve (beside your files, or on an
  OpenSCAD library path).

Then show how to add your library to a project — a `git clone`, a git submodule,
or copying the `.scad` files.
